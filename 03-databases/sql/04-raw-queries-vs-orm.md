# Raw Queries vs ORM

| Raw SQL               | ORM                   |
| --------------------- | --------------------- |
| Full control          | Convenience           |
| Performance optimised | Abstraction layer     |
| More boilerplate      | Less code             |
| Manual migrations     | Automatic schema sync |

Use raw queries for complex reporting; ORM for typical CRUD.
