# Migrations and Seeds

## Knex

- `knex migrate:make migration_name`
- `knex migrate:latest`
- `knex seed:run`

## Sequelize

- `sequelize migration:create`
- `sequelize db:migrate`
- `sequelize db:seed:all`

Keep database schema versioned.
